Support & Troubleshooting

Have questions or running into issues with Manather? Find answers and help resources below.

Frequently Asked Questions

Why isn't a screenshot generated for my web link?

When you add a web link, Manather spins up a headless WebKit render engine to capture a snapshot of the page. This process may take a few seconds. If the page fails to render, check your internet connectivity. Also note that some websites block automated screenshot requests or require complex login screens.

Where are my imported images physically stored?

For performance and sandbox safety, Manather copies all imported visual references into the application’s local Sandboxed Container folder under:
~/Library/Containers/com.manath.manather/Data/Documents/ (or Application Support).
Deleting references inside the app's Trash permanently removes these physical files from disk.

Does Manather require an active internet connection?

No. The core functionality—such as the gallery grid, color palette extraction, details inspector, and local search—operates completely offline. Network connectivity is only required to load and generate screenshots for newly imported web links.

How can I request a feature or report a crash?

Since Manather is open-source, the best place to submit bugs, feature ideas, or request code updates is on the repository issues tracker on GitHub.
