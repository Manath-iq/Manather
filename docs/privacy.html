Privacy Policy

Last Updated: June 11, 2026

This Privacy Policy describes how Manather ("we", "our", "the application") processes data when you install and run the application. We take your privacy very seriously and are committed to maintaining a completely safe and transparent user experience.

1. Local-First Data Processing (Zero Collection)

Manather is designed to be an entirely local-first, offline-capable database gallery.

  • No Accounts: You do not need to register, sign in, or create an account to use the application.
  • No Telemetry or Analytics: The application does not collect, log, or transmit usage statistics, error logs, or behavioral data to any remote server.
  • No Cloud Storage: All references, images, GIFs, snippets, titles, notes, and prompts remain locally stored on your machine. We do not use third-party cloud hosting or external sync providers in this version.

2. Sandboxed File Storage & Permissions

To ensure highest compliance with macOS security protocols, Manather operates securely within the OS Sandbox.

  • Application Sandbox: When you import an image or GIF file into the app, the file is physically copied and isolated inside the application's secure container (typically inside ~/Library/Containers/ or your system's Application Support folder).
  • Relative File Path Database: The local SwiftData database stores only the metadata and a relative file path pointing to the copied asset within the container. No absolute file paths or system-wide directory access tokens are tracked.

3. Headless Web View Operations

When you add a web link to catalog a layout reference:

  • The app creates a headless, on-device instance of WebKit (WKWebView) to load the specified page, render its contents, and capture a static snapshot.
  • This action behaves identically to opening the URL in Safari or another browser on your Mac. No analytical trackers, cookies, or credentials are captured or saved by the application during this snapshot generation.

4. Changes to This Privacy Policy

We may update this policy periodically to reflect updates to the app's functionality. Any changes will be published here inside the repository website.
